TaskJar Software

Description: A project management software that helps with management of tasks at work. Users can add projects, add project tasks and then they can track the task progress as they work, based on task urgency.

    Technologies Used

  • Django
  • Django-Rest-Framework
  • Postresql
  • DRF-simpleJWT
  • React
  • Redux
  • Bootstrap

    Features

  • Authentication & Authorization
  • Create a Project to Manage
  • Add project contributors
  • Create Project Tasks
  • Assign Tasks
  • See task Board
  • Mark task as In-Progress
  • Mark Task as Done
  • Delete Task
  • Edit & Project
  • View & Edit Profile
  • Admin: See all Users
  • Admin: Give admin status to users
  • Admin: See all Users

Things I Learnt

  1. How to 'RESET' state in the redux store whenever i need to go back to the default state
  2. User JWT for Authentication and Authorization in django
  3. How to effectively nest model serializers in django
  4. How to properly construct model relations in relational databases(Postgres)
