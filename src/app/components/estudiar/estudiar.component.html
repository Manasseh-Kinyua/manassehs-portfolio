Estudiar Web App

Description: A web application where users can create chat-rooms/study-rooms to help each other with learning their specific areas of interest.

    Technologies Used

  • Django
  • Django-Rest-Framework
  • Postresql
  • DRF-simpleJWT
  • React
  • Redux
  • Bootstrap
  • AWS-RDS

    Features

  • Authentication & Authorization
  • Guest user Login
  • Create a Study Room
  • Search & Filter Rooms(by multiple parameters)
  • Write a room review
  • Chat with room members
  • View room details
  • View recent activity
  • View room participants
  • Mark task as In-Progress
  • Delete a room that you created
  • Edit a room that you created
  • View & Edit Profile
  • View rooms you have created in your profile
  • Admin: See all Users
  • Admin: Give users admin status
  • Admin: Remove users
  • Admin: See all rooms

Things I Learnt

  1. How to host my Postgres database on AWS-RDS
  2. How to handle many-to-many relationships in relational databases
