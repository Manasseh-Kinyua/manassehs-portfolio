Ascendas Web App

Description: This is a web based asset management application that allows users to interact with Ascendas, a company that specializes in the fields of property & facility management, commercial real estate, agency, valuation and research.

Role: In a team of five, I worked as Frontend Developer building with Angular. Main roles involved designing and building a good Frontend to ensure good user experience. The role also involved active consumption of APIs built by our Backend developer.

    Technologies Used

  • Angular
  • Karma & Jasmine
  • Angular Material
  • Figma

    Features

  • Authentication and Authorization
  • Create different account types(client / landlord / manager / contractor)
  • Add different property and asset types
  • Request various services for your assets
  • Rent Payments
  • Payments for management services
  • Payment invoices
  • Make updates on maintenance progress
  • Manager: assign users contractor status

Things I Learnt

  1. Automated testing with karma and jasmine
  2. How to consume apis with angular
